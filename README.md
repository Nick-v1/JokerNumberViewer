A simple project for searching through the Greek's lottery Joker Results.

Created with:
- .net 10 for the backend
- react/vite/typescript for frontend for the frontend
- ClosedXML for parsing xml result files
- QuestPDF for creating the pdf file
- lucide-react for the icons
- ag-grid-community and ag-grid-react for the Grid used for displaying the results

Features:
- Grid
- Searching
- Filtering
- Pagination
- Search by Year (years included: 2020 - 2026)
- Result Export

Runs fully locally, no external calls.
